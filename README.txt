== Description ==

This plugin can block adblock installed users from accessing the site and can also display alternative ads served from the same domain.

To learn how this plugin detects adblock and displays alternative ads visit http://www.sitepoint.com/creating-an-anti-adblock-plugin-for-wordpress/

== Installation ==

1. Upload the alternative folder to the /wp-content/plugins/ directory.

2. Activate the plugin via the 'Plugins' menu in WordPress.

3. Plugin settings are located in "Anti Adblock".
